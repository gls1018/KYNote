### 矩阵



- 矩阵的乘法
  - AB 一般不等于 BA
  - AB等于0 且A不等于 0; 不能推出 B=0
  - AB等于AC 且 A不等于 0;  不能推出 B = C
  - A<sup>2</sup> = 0, 不能推出 A = 0;
  - **|AB| = |BA| = |A||B| 成立**
- 可逆矩阵
  - A*A<sup>\*</sup> = |A| * E
  - 矩阵A可逆的充分必要条件是 |A| != 0;
