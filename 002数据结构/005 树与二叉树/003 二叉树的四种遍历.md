- 线序遍历
  - 先Root
  - 后左右
- 中序遍历
  - 先左
  - 后Root
  - 最后右
- 后续遍历
  - 先左右
  - 后Root
- 层次遍历
  - 一层一层遍历
  - 代码实现时，需要借助一个队列

**若已知中序遍历，再加上其他三种遍历中的任意一种，就可以唯一确定一颗二叉树**

层次遍历算法思想：

- 1. 先将Root节点入队
- 2. 若队列非空，队头节点出队，访问该节点，若它有左孩子，则入队，若它有右孩子，则入队
- 3. 重复步骤2

层次遍历代码:

```cpp
void LevelOrder(BiTree T)
{
    InitQueue(Q); //初始化队列
    BiTree p; //中间变量，保存当前访问的节点
    EnQueue(Q, T);
    while(! IsEmpty(Q))
    {
        DeQueue(Q, p);
        visit(p);
        if(p->lchild != NULL)
            EnQueue(Q, p->lchild);
        if(p->rchild != NULL)
            EnQueue(Q, p->rchild);
    }
}
```
