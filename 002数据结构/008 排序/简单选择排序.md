## 1. 简单选择排序

基本思想:

- 每一次迭代, 比如第i次, 在n-i-1个待排序元素中, 选择一个最小的元素作为有序子序列的第i个元素.
- 进行n-1次迭代, 只剩下1个元素, 一定是最大的元素, 此时整个序列有序.

```cpp
void SelectSort(int A[], int n)
{
    for(int i=0; i<n-1; i++)
    {
        int min = i;
        for(inr j = i+1; j<n; j++)
        {
            if(A[j]<A[min])
                min = j;
		}
        if(min!=j)
            swap(A[i], A[min]);
    }
}
```

- 空间效率: 使用常数个辅助单元, 空间复杂度为O(1).

- 时间效率:

- 算法稳定性: 
  - 不稳定排序
  - 比如序列 {2, <span style="color:red">2</span>, 1}, 在经过第一次排序后, 变成 {1, <span style="color:red">2</span>, 2}, 最终结果也是{1, <span style="color:red">2</span>, 2}.

- 适用场景:
  - 顺序存储或者链式存储的线性表.
