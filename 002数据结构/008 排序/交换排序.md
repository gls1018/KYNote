## 1. 冒泡排序

算法思想:

- 从前往后或者从后往前, 两两比较相邻元素的值.

- 如果 A[i] > A[i+1]; 则交换它们.
- 这样每一轮下来, 最大的元素出现在末尾.
- n个元素需要n-1次迭代, 不需要n次, 因为n-1次时, 只剩下1个元素了.

```cpp

void BubbleSort(int Array[], int n)
{
    for(int i=0; i<n-1; i++)
    {
        // flag的作用是记录本次迭代是否有元素交换
        // 如果没有, 说明元素已经有序,可以直接break;
        bool flag = false;
        for(int j=0; j<n-i-1; j++)
        {
            if(Array[j+1]<Array[j])
            {
                Swap(Array[j+1], Array[j]);
                flag = true;
            }
        }
        if(flag == false)
            break;
	}
    return;
}
```



性能分析:

- 使用了常数个辅助单元, 空间复杂度为O(1).
- 时间效率:
  - 当前序列有序, 需要1次迭代， flag为false, 直接break, 比较次数为n-1， 移动次数为0， 时间复杂度为O(n).
  - 当前序列为逆序，需要n-1次迭代. 每次迭代, 需要进行 n-i次元素交换
    - 比较次数 $= \sum_{i=1}^{n-1}{}{(n-i)} = \dfrac{n(n-1)}{2}$
    - 元素交换次数 = $\dfrac{n(n-1)}{2}$
    - 元素移动次数 = $\dfrac{3n(n-1)}{2}$



这里说一下为什么元素移动次数是交换次数的三倍.

因为swap函数里面交换两个元素需要进行三次赋值操作

```cpp
void swap(int &a, int &b)
{
    int temp = a;
    a = b;
    b temp;
}
```





- 冒泡排序是稳定排序算法.
- 冒泡排序适用于顺序存储和链式存储的线性表.



## 2. 快速排序

适用性: 快速排序仅仅适用于顺序存储的线性表.

```cpp
int Partition(int array[], int low, int high)
{
	int pivot = array[low];
	while (low < high)
	{
		while (low < high && array[high] >= pivot)
			--high;
		array[low] = array[high];
		while (low < high && array[low] <= pivot)
			++low;
		array[high] = array[low];
	}
	array[low] = pivot;
	return low;
}


void QuickSort(int array[], int low, int high)
{
	if (low < high)
	{
		int pivotpos = Partition(array, low, high);
		QuickSort(array, low, pivotpos - 1);
		QuickSort(array, pivotpos + 1, high);
	}
}
```

稳定性: 

- 不稳定排序

- 比如: 3, **2**, 2, 选择3作为pivot,  最终结果为 2, **2**， 3

空间复杂度:

- 需要一个递归工作栈来保存每次递归调用的必要信息, 最好情况为 O(log~2~n), 最坏情况为 O(n), 平均栈深度为 O(log~2~n)

时间复杂度:

- 最坏情况为 O(n^2^).

- 最好情况是 O(nlog~2~n).

快速排序是所有内部排序算法中平均性能最优的排序算法.
