# 2. 数据的表示和运算

## 2.1 数制与编码

- 真值：符合人类习惯的数字
- 机器数：数字实际存到机器里的形式
- 定点数：小数点的位置固定
- 浮点数：小数点的位置不固定

------

正数的原码、反码、补码符号位相同，机器码相同。移码只能表示正数。

[x]<sub>原</sub> --> [x]<sub>反</sub>： 除符号位外，全部取反

[x]<sub>原</sub> --> [x]<sub>补</sub>：除符号位外，全部取反，末位加一

[x]<sub>补</sub> --> [x]<sub>原</sub>： 除符号位外，全部取反，末位加一

[x]<sub>补</sub> --> [x]<sub>移</sub>：符号位取反

[x]<sub>反</sub> --> [x]<sub>补</sub>：末位加一，注意进位

[x]<sub>补</sub> --> [-x]<sub>补</sub>：符号位、数值位全部取反，末位加一。

------

原码和反码有 +0 和 -0 之分：

``` 
// 原码的 +0 和 -0
0000 0000
1000 0000

// 反码的 +0 和 -0
0000 0000
1111 1111

//补码的 0
0000 0000
0000 0000
```

可见，补码的 0 只有一种表示状态， 补码的 1000 0000 表示 -2<sup>7</sup>
